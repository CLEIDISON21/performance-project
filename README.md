# Performance Project (Node.js + k6)

Projeto de testes de performance e stress utilizando Node.js e k6.

## Objetivo
Simular carga em uma API e analisar desempenho.

## Como rodar
npm install  
npm start  
npm run performance  
npm run stress  

##  Resultados
- Até 500 usuários → estável  
- 1000+ → degradação  
- 1500 → alta latência  
- 0% de falhas  

## Tecnologias
- Node.js  
- Express  
- JWT  
- k6
